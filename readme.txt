Useful Plugin to Display Skype Buttons and Online Skype Status Through Widgets.

== Description ==
<p>
This WordPress Plugin Allows You to Add one Skype Widget to Your Site's Sidebar. You Can Pick any One of The Available Skype Button Colors i.e. Green & Blue.

This is Very Useful Tool to Display Skype Chat Badge to your Site. It will helps Your Visitors to Connect with you Over Skype Easily. Its Not only limited to add Chat feature But also you can easily Display few More Skype Options like Display Call Options, Chat Option, Display User Info, Send Voice Mail, Send File etc... As per your Need you Can set which Badge to Display over your Site.
</p> 

== Installation ==

1. Download Archive & Unpack It.
2. Upload (& overwrite) the /MySkype/ folder & its content to the /plugins/ folder. 
3. Activate the plugin in your Plugin Administration page.
4. Drag the "My Skype Widget" on your Sidebar.
5. Configure your "My Skype Widget".


== Configuration ==
1. Set the Title
2. Set the User ID
3. Select the Buttons you like to Use.
4. Show Call: Show the Call Me Button.
5. Show Add:  Show the Add Me to Skype Button.
6. Show Chat: Show the Chat With Me Button.
7. Show Info: View My Profile Button.
8. Show Voicemail: Show the Leave Me a Voicemail Button.
   You need an additional Skype Subscription for Receiving Voicemails, please Check <a href="http://www.skype.com/en/features/">Voicemail Subscription</a>
9. Show Sendfile: Show the Send Me a File Button.
10. Select the Color Option (Green or Blue)


== Frequently Asked Questions ==

= My Online Status is Not Displaying? =
In Order to Activate the Online Status you Need to Configure Your <a href="http://www.skype.com/share/buttons/status.html">Skype Account</a> to Allow it.

= I Can't Receice any Voicemail =

You Need an Additional Skype Subscription For Receiving Any Voicemails Please Check <a href="http://www.skype.com/en/features/">Voicemail Subscription.</a>
